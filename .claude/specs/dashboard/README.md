# Dashboard specs

Per-page behavior specs owned by the **Dashboard** engineering team.

## What this team is delivering

Integrating the new page header, dashboard toolbar, applied filters bar, and saved-view integration into the existing production analytics dashboards. Chart data is already in production — engineering adds the chrome.

## Specs in this folder

- `specs-service-overview.md` — canonical reference page (read first — all others follow this pattern)
- `specs-comparison-users.md` — stats bar + Notify Users button
- `specs-comparison-categories.md` — same shape as comparison-users
- `specs-comparison-topics.md` — same shape as comparison-users
- `specs-custom-reports.md` — single dashboard, no tabs, no stats bar
